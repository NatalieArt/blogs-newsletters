Agent, Guided, and Manual modes now sit inside one creation flow. Start with a conversation, let the assistant resolve the technical setup, then keep going into image and video without leaving the same workspace.

Meet the modes

Three ways to create, without locking you into one workflow.

The original article introduces the new Creative Assistant as a workflow choice rather than a single interface. You can stay in Agent mode for a chat-first experience, work in Guided mode when you want chat and controls visible at the same time, or drop into Manual mode when precision matters more than assistance.

Agent modeBest when you want to describe the goal in plain language and let the assistant shape the setup.
Guided modeKeeps the conversation visible beside the controls, so you can steer while still getting help.
Manual modeReturns you to the classic hands-on workflow whenever you want direct control.

That flexibility is what makes the tool feel native to the product instead of bolted on top of it.

The value prop

Start with a conversation, then tighten the workflow only where needed.

“The assistant can suggest tasks like analyzing an image, enhancing a prompt, creating an article cover, transferring motion to a character, or answering product questions before you even type.”

Step 1

Attach media and the assistant immediately starts making sense of it.

One of the strongest parts of the original walkthrough is how quickly the assistant turns an image into actionable context. You attach one image, and the tool reads back the subject, setting, expression, and scene in plain language before suggesting what to do next.

What it sees

The subject, composition, mood, and scene context without a separate analysis prompt.

What it suggests

Ready-made next steps like prompt enhancement, scene edits, or creative direction changes.

What you decide

Which branch to take and whether to stay conversational or move toward direct controls.

Step 2

Ask for the edit in plain language and let the assistant resolve the technical settings.

The workflow shown in the source article is intentionally simple: ask for the change in ordinary language, then let the assistant figure out the underlying setup. That includes model choice, settings, and the logic of the next operation.

Plain-language directionDescribe the edit you want instead of translating everything into a technical parameter set first.
Background resolutionThe assistant handles model selection, settings, and the hidden wiring of the task.
Human approval stays centralYou still review the output and decide whether to continue, refine, or switch modes.
Step 3

Once the image works, continue into video without restarting the workflow.

The article uses a strong example here: edit an image first, then apply a second prompt to animate the same result into a speaking clip. That continuity matters. The assistant is not just a prompt box; it helps carry creative intent across media types.

While the job runs, you can inspect Activity to see what the agent is doing in the background. If you want more control, you can shift into Guided or Manual mode without abandoning the work already in progress.

Next move

Use the assistant when momentum matters, then take the wheel whenever precision matters more.

The main promise of the new assistant is not total automation. It is continuity. You can begin in natural language, let the tool handle the first technical layer, and still reclaim direct control whenever the creative work gets specific.
